me: private는 밖으로 내보내지 않을 것들

## static
**Parang**: 템플릿, HTML DOM 그릴 때 사용한다. / 사용이 편하도록 static을 사용하기도 한다. 예를 들어 `Array.isList()`를 사용할 때마다 `new Array()`를 할 수 없기에 static으로 지정해두었다.
